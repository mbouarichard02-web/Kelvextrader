## 🚀 Démarrage Rapide

### 1. Créer un Compte

1. Cliquez sur **"Connexion"** en haut à droite
2. Authentifiez-vous avec Manus OAuth
3. Complétez votre profil utilisateur

### 2. Accéder au Dashboard

Après connexion, vous accédez automatiquement au **Dashboard Principal** avec :
- ✅ Statut algorithmique en direct
- ✅ Sentiment meter du marché
- ✅ Métriques de trading
- ✅ Accès au chat IA

### 3. Commencer à Trader

1. Consultez les **Analyses IA** pour les paires EUR/USD, V75, V100
2. Utilisez le **Chat IA** pour poser vos questions
3. Débloquez les **Modules Éducatifs** en augmentant votre score

## 📈 Analyse de Marché

### Accéder aux Analyses

1. Cliquez sur une **Paire de Trading** (EUR/USD, V75, V100)
2. Consultez l'**Analyse IA Détaillée**
3. Vérifiez les **Signaux Smart Money**

### Comprendre les Signaux

#### **Order Blocks**
Zones où les institutions accumulent/distribuent

#### **Liquidité**
Zones où le prix va chercher les stops

#### **Niveaux Clés**
Support/Résistance majeurs identifiés par l'IA

### Utiliser les Analyses

✅ **Pour les Traders Débutants:**
- Suivez les recommandations IA
- Utilisez les niveaux clés comme entrée/sortie
- Respectez les niveaux de risque suggérés

✅ **Pour les Traders Avancés:**
- Combinez les analyses avec votre stratégie
- Utilisez les signaux Smart Money pour confirmer
- Ajustez les paramètres selon votre style

## ❓ FAQ

### Questions Générales

**Q: Comment fonctionne KelvexTrader?**
A: KelvexTrader combine l'analyse de marché en temps réel, les concepts Smart Money et l'IA pour fournir des recommandations de trading intelligentes.

**Q: Dois-je avoir de l'expérience en trading?**
A: Non! KelvexTrader est conçu pour tous les niveaux. Les modules éducatifs vous guident du débutant à l'expert.

**Q: Puis-je utiliser KelvexTrader sur mobile?**
A: Oui! KelvexTrader est entièrement responsive et fonctionne sur tous les appareils.

### Questions Techniques

**Q: Comment fonctionne le Chat IA?**
A: Le Chat IA utilise des modèles de langage avancés avec le contexte de marché en direct pour fournir des réponses pertinentes et actualisées.

**Q: Les analyses sont-elles en temps réel?**
A: Oui! Les analyses se mettent à jour toutes les minutes avec les dernières données de marché.

**Q: Comment puis-je exporter mes analyses?**
A: Cliquez sur le bouton **"Exporter"** sur n'importe quelle analyse pour télécharger en PDF ou CSV.

### Questions de Paiement

**Q: Quelles sont les méthodes de paiement acceptées?**
A: Nous acceptons les cartes bancaires, Mobile Money (Orange, MTN, Wave) et les cryptomonnaies.

**Q: Puis-je annuler mon abonnement?**
A: Oui, vous pouvez annuler à tout moment depuis vos paramètres. Aucune pénalité.

**Q: Y a-t-il une période d'essai gratuite?**
A: Oui! Vous avez accès à la version Free avec des analyses limitées.

### Questions de Sécurité

**Q: Mes données sont-elles sécurisées?**
A: Oui! Nous utilisons le chiffrement SSL/TLS et les meilleures pratiques de sécurité.

**Q: Comment puis-je protéger mon compte?**
A: Activez l'authentification à deux facteurs dans vos paramètres de sécurité.

**Q: Que se passe-t-il si j'oublie mon mot de passe?**
A: Cliquez sur **"Mot de passe oublié"** sur la page de connexion pour réinitialiser.
